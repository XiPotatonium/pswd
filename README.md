# pswd

## Usage

### add

Repeatedly add record. Enter empty site to stop adding.

```>>> add```

### del

Delete certain record

```>>> del [index]```

### ls

List records

```>>> ls```

### mod

### import

Import from .tsv file

```import [file-path]```

### export

Export to .tsv file

```export [file-path]```

### chpswd

Change password.

```>>> chpswd```

### q

Exit program.

```>>> q```
